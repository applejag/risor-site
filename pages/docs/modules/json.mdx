# json

## Functions

### marshal

```go filename="Function signature"
marshal(v object) string
```

Returns a JSON string representing the given value.

```go copy filename="Example"
>>> m := {one: 1, two: 2}
>>> json.marshal(m)
"{\"one\":1,\"two\":2}"
```

### unmarshal

```go filename="Function signature"
unmarshal(s string) object
```

Returns the value represented by the given JSON string.

```go copy filename="Example"
>>> json.unmarshal("{\"one\":1,\"two\":2}")
{"one": 1, "two": 2}
```

### valid

```go filename="Function signature"
valid(s string) bool
```

Returns whether the given string is valid JSON.

```go copy filename="Example"
>>> json.valid("42")
true
>>> json.valid("{oops")
false
```
