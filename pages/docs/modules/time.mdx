# time

## Attributes

### RFC3339

```text filename="Example"
>>> time.parse(time.RFC3339, "2023-08-07T21:19:27-04:00")
2023-08-07T21:19:27-04:00
```

## Functions

### now

```go filename="Function signature"
now() time
```

Returns the current time.

```text copy filename="Example"
>>> time.now()
2023-08-07T21:19:27-04:00
```

### parse

```go filename="Function signature"
parse(layout, value string) time
```

Parses a string into a time.

```text copy filename="Example"
>>> time.parse(time.RFC3339, "2023-08-07T21:19:27-04:00")
2023-08-07T21:19:27-04:00
```

### since

```go filename="Function signature"
since(t time) duration
```

Returns the time elapsed since the given time.

```text copy filename="Example"
>>> time.since(time.parse(time.RFC3339, "2023-08-01T12:00:00-04:00"))
duration(153h24m34.821188s)
```
